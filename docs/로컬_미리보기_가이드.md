# 수정한 화면 로컬에서 미리보기 (VS Code)

이 프로젝트는 `src/pages/` 안의 원본 파일을 고친 다음, 한 번 "빌드"해야 실제 화면으로 볼 수 있어요. 아래 순서대로 하면 됩니다.

## 0. 최초 1회만 준비

### Node.js 설치 확인
PowerShell(또는 VS Code 터미널)에서:
```
node --version
```
버전이 안 뜨면 설치:
```
winget install --id OpenJS.NodeJS.LTS -e --source winget
```
설치 후 터미널(또는 VS Code)을 완전히 닫았다가 다시 열어야 인식됩니다.

### VS Code에 Live Server 확장 설치
왼쪽 사이드바 Extensions 아이콘 → "Live Server" 검색 → 설치 (Ritwick Dey 제작)

## 1. 편집

1. VS Code에서 **프로젝트 전체 폴더**(`ai-leaders-site`)를 엽니다. (File → Open Folder)
2. `src/pages/...` 안에서 원하는 페이지 파일을 찾아 수정합니다.
   - 예: 홈페이지 → `src/pages/index.html`
   - 예: 소개 페이지 → `src/pages/company/about.html`
3. 저장합니다 (Ctrl+S).

**주의**: `dist` 폴더 안의 파일은 절대 직접 수정하지 마세요. 아래 빌드 과정에서 매번 통째로 새로 만들어지는 결과물이라, 거기 손댄 내용은 사라집니다.

## 2. 자동 빌드 감시 켜기 (최초 1회만 실행하면 계속 켜둠)

같은 창의 터미널(Ctrl + ` 로 열기)에서:
```
node tools/watch-build.mjs
```
`src/` 안의 아무 파일이나 저장할 때마다 이 스크립트가 자동으로 다시 빌드해 줍니다. 수동으로 빌드 명령어를 반복할 필요가 없어요. 이 터미널 탭은 계속 켜둔 채로 두면 됩니다. (끄고 싶으면 Ctrl+C)

## 3. 미리보기 (최초 1회만 창을 따로 준비)

1. **File → New Window**로 새 VS Code 창을 하나 더 엽니다.
2. 그 새 창에서 **File → Open Folder** → 이번엔 `ai-leaders-site` 안에 있는 **`dist` 폴더 자체**를 선택해서 엽니다. (전체 폴더가 아니라 `dist`만!)
3. 왼쪽 탐색기에서 `index.html`을 오른쪽 클릭 → **"Open with Live Server"**
4. 브라우저가 자동으로 열리거나, 안 열리면 직접 주소창에 `localhost:5500` 입력

이후로는 이 미리보기 창은 그대로 켜두고, 편집 창에서 파일 수정 → 저장만 하면 됩니다. `watch-build.mjs`가 자동으로 빌드하고, Live Server가 자동으로 새로고침합니다.

## 요약 흐름

```
[편집 창]                    [빌드 감시 터미널]              [미리보기 창]
src/pages/*.html 수정 → 저장
      ↓
                        자동 감지 → 자동 빌드
                              ↓                        자동 새로고침
```

## 자주 겪는 문제

- **`Cannot GET /` 에러**: `dist` 폴더가 아니라 다른 폴더(예: 프로젝트 루트)를 Live Server로 열었을 때 나는 에러입니다. 반드시 `dist` 폴더 자체를 열어야 합니다.
- **"Go Live" 눌러도 안 켜짐 / 새 창 안 뜸**: 포트가 이미 사용 중이거나 확장이 꼬인 경우입니다. 컴퓨터를 재시작한 뒤 다시 시도하면 대부분 해결됩니다.
- **수정했는데 미리보기에 반영이 안 됨**: `node tools/watch-build.mjs`를 켜둔 터미널이 닫혀 있진 않은지 확인하세요. 닫혀 있었다면 다시 실행하면 됩니다.
- **`node tools/build-cloudflare-pages.mjs`를 직접 한 번만 실행하고 싶을 때**: `watch-build.mjs` 대신 이 명령어를 그때그때 직접 쳐도 됩니다 (자동 감시 없이 1회성 빌드).
